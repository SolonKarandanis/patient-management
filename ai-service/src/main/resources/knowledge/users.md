# User Management

## What is user management?
User management allows administrators to search for, view, and manage user accounts in the system. It is accessible from the Users section in the left sidebar.

## How to search for users
Navigate to Users in the sidebar. The search form at the top lets you filter by username, name, email, status, and role. Fill in one or more fields and click Search. Results are displayed in the table below the form. You can sort the table by clicking column headers.

## How to view user details
From the search results table click on a user row or the details icon to open the user details page. The details page shows the user's full profile including their assigned roles, permissions, and account status.

## User account status
Users can be active or inactive. Inactive users cannot log in. An administrator can activate or deactivate a user from the user details page.

## Roles and permissions
Each user is assigned one or more roles. Roles group together a set of permissions that control what the user can see and do in the application. Permissions are granular — for example MANAGE_RESOURCE_BUNDLES grants access to the Resource Bundles feature. Roles and permissions are managed by the system administrator.

## Changing a password
A user can change their own password from their profile page. Administrators can reset another user's password from the user details page.

## Exporting users
The user search results can be exported to CSV. After executing a search click the Export button to download the results as a CSV file.
